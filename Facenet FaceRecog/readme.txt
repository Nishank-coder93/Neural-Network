#########################################
	     PRE - REQS
#########################################

- First download the pre-trained model from the link given below

https://drive.google.com/file/d/0B5MzpY9kBtDVZ2RpVDYwWmxoSUk/

- Unzip the Downloaded file
- Place the unzipped folder named "20170512-110547" in "src" folder as it is

#########################################
	     STARING UP
#########################################

- Navigate to project folder
- "python3 cvstart.py" to start the application in Python 3
- The application may take upto "3 min" to start as it loads the Weights and preTrained model in the background. 
- Once loaded a camera window can be seen and multiple faces can now be detected
- The detected image and the Actual predicted image is shown on the side as comparison from the trained images folder Face_data_compare
